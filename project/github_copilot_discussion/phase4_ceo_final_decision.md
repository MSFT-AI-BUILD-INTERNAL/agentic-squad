## 4. 전사 확대 Go/No-Go 기준

### ✅ Go 조건 (모두 충족 시)
- 6개월 파일럿 ROI ≥ 96% (보수적 기준)
- Critical/High 보안 인시던트 **누적 0건**
- 개발자 NPS ≥ 40
- CI 통과율 85% 이상 지속 유지

### 🛑 No-Go 조건 (하나라도 해당 시)
- 4개월 차 ROI < 80% → **범위 축소**
- Critical/High 인시던트 1건 이상 → **Agent 즉시 비활성화, 원인 분석 후 재개 판단**
- 개발자 NPS < 20 → **파일럿 중단 검토**
